# VueJs-Auth-Firebase
Este projeto será desenvolvido em Vue.js utilizando o sistema de autenticação do Firebase.

## Recursos implementados
* Rotas de navegação com "vue-router"
* Integração com Firebase




## Project setup
```
npm install
```
